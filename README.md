# Chinese Exam Trainer

Static web app for the A1.1 course material, Units 1–5.

## Files
- `index.html`
- `style.css`
- `data.js`
- `app.js`
- `manifest.json`
- `sw.js`

## Local use
Open `index.html` in a browser. For full PWA/offline support, serve the folder through a local web server or GitHub Pages.

## GitHub Pages
1. Upload all files to the repository root.
2. Open **Settings → Pages**.
3. Select **Deploy from a branch**.
4. Choose `main` and `/ (root)`.
5. Save and wait for deployment.
